# ShufflePic アイコン画像説明

## デザイン概要

複数の画像をシャッフルして表示する ShufflePic の機能を、3枚の写真カードと交差する2本の矢印で表現したアプリアイコンです。

- 背面には紫、緑、青の3枚の写真カードを重ねています。
- 写真カードは、画像フォルダ内に複数の画像が存在することを示します。
- 前面の青い交差矢印は、画像のシャッフル再生と順序の入れ替えを示します。
- 青を中心とした配色により、Windowsアプリとして視認しやすい、明瞭で落ち着いた印象にしています。
- 白い太めの縁取りと単純化した輪郭により、16pxや32pxの小さな表示でも形状を判別しやすくしています。
- 文字、製品名、透かしは画像内に含まれていません。

## 代替テキスト

短い説明：

> 3枚の写真に青いシャッフル矢印を重ねたShufflePicのアイコン

詳しい説明：

> 紫、緑、青の3枚の写真カードが扇状に重なり、その前面を2本の青い矢印が交差する、画像のシャッフル再生を表したアイコン。

## ファイル

| ファイル | 仕様 | 主な用途 |
| --- | --- | --- |
| `shufflepic-icon-1024.png` | 1024×1024、RGBA、背景透過 | アイコンのマスター画像、ストア素材、派生画像の作成 |
| `shufflepic-icon-512.png` | 512×512、RGBA、背景透過 | README、Webページ、配布ページ |
| `app.ico` | 16 / 32 / 48 / 256px内包、背景透過 | Windows実行ファイル |

## 制作・加工方針

元のアイコン案を基に、アプリアイコン用途へ次の調整を行っています。

- 構成要素を「複数の写真」と「シャッフル矢印」に限定
- 小サイズで潰れやすい細部、過剰な陰影、複雑な紙の折り目を簡略化
- アイコン外周に安全な余白を確保
- 背景を完全に透過
- 紫色の写真カードを損なわずに透過処理できるよう、橙色のクロマキー背景を経由して加工
- PNGからWindows用マルチサイズICOを生成

## 使用上の注意

- マスター画像は `shufflepic-icon-1024.png` とします。
- サイズ変更時は縦横比を固定し、変形させないでください。
- アイコンの一部を切り取ったり、矢印と写真カードを別々に配置したりしないでください。
- 背景色を付ける場合は、白い外周が判別できる中間色または濃色を推奨します。
- 極端に明るい背景では白い縁取りが弱く見えるため、必要に応じて表示側で余白や背景プレートを設けてください。
- GitHub READMEでは、通常 `shufflepic-icon-512.png` を表示用として使用します。

## 生成情報

AI画像生成によって元案をアプリアイコン向けに再構成し、その後、背景透過、縮小、ICO変換を行っています。公開先の規約でAI生成画像に関する申告が必要な場合は、この点を確認してください。
