# ShufflePic ドキュメント索引

ShufflePic は、指定フォルダ直下の画像を **1〜2 枚並べてランダム（無重複）に自動送り**する写真スライドショー
アプリ（Windows / Rust + eframe・egui）です。公開バージョンは **v1.0**。実装本体は `../v1.0/`。

> デコードは別スレッド化（高速）。設定（フォルダ・間隔・表示枚数）とサイクル進行は**ディスク永続化**し、
> 同じフォルダなら次回起動で続きから。原本は読み取り専用で、削除は `delete`、巨大画像は `oversized` へ
> **移動**で退避（Undo は無し）。

## 開発クイックスタート

```
cd ../v1.0
cargo run --release      # 実行（初回は ⚙ 設定 でフォルダ選択）
cargo test               # ユニットテスト（13 件）
cargo clippy --all-targets -- -D warnings
cargo build --release
```

- 状態は `shufflepic_state.json`（実行ファイル隣／`%APPDATA%\ShufflePic` 等へフォールバック）に保存。
- GUI 実機動作は開発者が確認済み（実装報告 §6）。MSIX パッケージ版もローカル install で動作確認済み
  （公開計画書 B5）。公開準備は `8`（掲載情報）参照。
